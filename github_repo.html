These are the steps to push a github repository

First let's discuss what is git?
Git is a free-and-open-source distributed control system designed to handle everything from small to very large projects with ✨speed and efficiency✨.

Here are the steps to push your code to your own repository
(Assume that have Git installed on your computer).

  1. Create a repository either from scratch or from existing repository. (Mine have already created a repo already. I named that repo as webdev.)
  2. Initialize your created repository.
    • Go into the folder/directory containing the particular project.
    • Right-click that directory and select "Git Bash here..." from the right-click menu (this applied to a Windows operating system).
    • As Git Bash window opens, type git init.
    • Type git add to add all of your relevant files.
      Example: git add README.md
    • Commit your added file by typing git commit (I recommend including your commit message. Type: git commit -m "your commit message").
  3. Connect to your GitHub repo.
